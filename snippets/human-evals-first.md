# Human Evals First

## Early Eval Rubric

Before building automated evals, start with a human scoring sheet:

- Was the user intent interpreted correctly
- Was the returned answer or recommendation actually useful
- Did the explanation make the result believable
- Would the user take action based on this output
- What specific failure mode occurred

## Why Start Here

Human evals are the fastest way to discover whether the product is solving the right problem. Automated evals become more valuable after the team knows what good looks like.
